## Langkah - Langkah menjalankan program Aplikasi Country ReactJs:

## 1.Download Node Js Link Download [NodeJS](https://nodejs.org/en/download/prebuilt-installer/current)

## 2.Install Node.js Yang Sudah Di Download

## 3.Lalu clonnig program aplikasi country ReactJs dengan perintah`git clone https://github.com/nurmuhamadsoleh/TestFrontendCountry.git`

## 4. Jika terdapat *folder node module* harap dihapus terlebih dahulu, jika tidak ada *folder node module* bisa menjalankan perintah pada point 5.

## 5. Jika sudah selesai download / clonning, lalu jalankan perintah`npm install` Perintah untuk menginstall Node Module

## 6. Jika Sudah Maka Jalankan Program Aplikasi Country dengan mengetikan perintah`npm run dev` Dan Buka Browser Anda dan kunjugi url berikut ini http://localhost:8080
